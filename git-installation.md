## Gitをインストールしよう！

### その前にすでにインストールされていないかチェック

```bash
git --version
```

#### バージョンが表示されたらすでにインストール済み

> **Note**
> macOSの場合

```bash
git version 2.32.1 (Apple Git-133)
```

### いざインストール！

#### Ubuntu

```bash
sudo apt install git
```

#### Homebrew

> **Note**
> Homebrewのインストールが必要

```bash
brew install git
```
